Installation Instructions

  1. Make sure you have Python installed on your computer. You can download it from python.org.
  2. Open Command Prompt by searching for "cmd" in the Start menu.
  3. Clone the repository using Git: 
    git clone https://github.com/prai-git/repository-name.git
  4. Navigate to the game directory: 
    cd repository-name
  5. Install the required packages: 
    pip install -r requirements.txt
  6. Run the game: 
    python main.py
    (Replace main.py with the actual file name if different)

  1. Make sure you have Python installed on your computer. Most Macs come with Python pre-installed, but you can download the latest version from python.org.
  2. Open Terminal (Applications > Utilities > Terminal).
  3. Clone the repository using Git: 
    git clone https://github.com/prai-git/repository-name.git
  4. Navigate to the game directory: 
    cd repository-name
  5. Install the required packages: 
    pip3 install -r requirements.txt
    Note: On Mac, you might need to use pip3 instead of pip.
  6. Run the game: 
    python3 main.py
    (Replace main.py with the actual file name if different)
